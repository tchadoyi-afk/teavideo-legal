Politique de confidentialité

Application : Teavideo
Dernière mise à jour : 30 juin 2026

Teavideo est un outil personnel d'automatisation qui prépare de courtes vidéos et les téléverse vers le compte TikTok de son propre utilisateur, via l'API officielle « Content Posting » de TikTok. La présente politique explique quelles données sont utilisées et comment.

1. Données auxquelles l'application accède

  • Informations de base du profil TikTok (identifiant ouvert « open_id », nom d'affichage, avatar) via le scope user.info.basic, uniquement pour identifier le compte autorisé.
  • Téléversement de vidéos via le scope video.upload (ou video.publish), pour envoyer les vidéos créées par l'utilisateur sur son propre compte.

2. Comment les données sont utilisées

Les jetons d'accès et de rafraîchissement TikTok sont stockés localement sur l'ordinateur de l'utilisateur, et servent exclusivement à publier les vidéos de l'utilisateur sur son propre compte. Aucune donnée n'est revendue, partagée avec des tiers, ni utilisée à des fins publicitaires.

3. Conservation et suppression

Les jetons sont conservés tant que l'utilisateur souhaite utiliser l'application. Il peut les supprimer à tout moment en retirant l'autorisation depuis les paramètres de son compte TikTok et en effaçant le fichier de jetons local. Aucune base de données distante n'est utilisée.

4. Partage avec des tiers

Aucun. Les seules communications réseau ont lieu directement avec les serveurs officiels de TikTok pour la publication.

5. Sécurité

Les jetons sont stockés dans un fichier local sur la machine de l'utilisateur. L'utilisateur est responsable de la sécurité de son propre appareil.
